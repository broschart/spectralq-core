Radiation Monitoring

Sources: BfS ODL (Germany) + EURDEP (Europe)

Monitors ambient gamma dose rate in an area. BfS ODL provides data for ~1,600 stations in Germany, EURDEP for ~37,000 stations Europe-wide. Elevated values (>0.3 µSv/h) are flagged as anomalies.

Draw observation zones on the map. Gamma dose rate data will be fetched automatically for all stations in the zone.
